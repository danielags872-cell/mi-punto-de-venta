# mi-punto-de-venta

# 📦 Sistema de Inventario y Cuadre de Caja

Este es un sistema ligero diseñado para gestionar ventas diarias y control de existencias directamente desde el navegador.

### 🚀 Funcionalidades
* **Gestión de Inventario:** Registro de productos con precio y stock inicial.
* **Control de Ventas:** Descuento automático de existencias al realizar una venta.
* **Cuadre de Caja:** Sumatoria automática de los ingresos del día.
* **Persistencia Local:** Los datos se guardan en el navegador (Local Storage).

### 🛠️ Tecnologías utilizadas
* HTML5 / JavaScript (ES6)
* Tailwind CSS (Para el diseño responsivo)
